# sistema-di-gestione-delle-spese-personali-e-del-budget
Progetto finale di programmazione: sistema di gestione delle spese personali e del budget

QUICK START

Per avviare rapidamente il programma seguire questi passaggi:

Aprire la cartella principale del progetto:

Sistema di gestione delle spese personali e del budget

Entrare nella cartella:

"src"

Individuare il file:

"main.py" (scaricare il file)

"spese.db" (scaricare il file)

Cliccare con il tasto destro del mouse sul file "main.py".

Selezionare:

Edit with IDLE

Una volta aperto l'editor Python premere il tasto F5.

Il programma verrà eseguito nella Python Shell, che rimarrà aperta per permettere l'interazione con il sistema.

MENU DEL PROGRAMMA

Dopo l'avvio comparirà il seguente menu:

GESTIONE SPESE PERSONALI

1 - Gestione Categorie

2 - Inserisci Spesa

3 - Definisci Budget Mensile

4 - Visualizza Report

5 - Esci

Digitare il numero corrispondente all'operazione desiderata.
